# 2022Meetups

A collection of slide decks and presentation materials from BBSW meetups held in 2022.

## Folder naming convention

Each folder is named `<TOPIC>_<MMDD>`, where `<TOPIC>` is a short abbreviation for the meetup subject and `<MMDD>` is the month and day of the meetup.

Examples:
- `CID_0317/` — meetup on March 17 focused on CID
- `BICR_0526/` — meetup on May 26 focused on Blinded Independent Central Review (BICR)

## Contributing

To add materials for a new meetup:

1. Create a folder following the `<TOPIC>_<MMDD>` convention (e.g., `XYZ_1103/`).
2. Place the relevant PDFs or slide decks inside the folder.
3. Use clear, descriptive file names so the content is easy to identify.
4. Open a pull request against `main`.
